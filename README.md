# WB Примерочная

Лендинг для Telegram-бота виртуальной примерки одежды.

## Стек технологий
- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)

## Начало работы
1. Установите зависимости: `npm install`
2. Запустите сервер разработки: `npm run dev`
3. Откройте [http://localhost:3000](http://localhost:3000)

## Деплой
Проект настроен для автоматического деплоя на [Vercel](https://vercel.com/).
